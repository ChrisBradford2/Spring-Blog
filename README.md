
# Spring-blog

Projet de fin de module en Développement API

## Demo

https://spring-blog-client.herokuapp.com/

## Requirements

- npm
- Java JDK 11
- MySQL

### Mysql

- MySQL on localhost:3306
- Credentials :
  - username=root
  - password=root

## Pourquoi React ?

- Plus de simplicité
- Plus adaptatif

## Installation

Install Spring-blog with npm

```bash
  npm install
```

## Run Locally

Start the server

```bash
  npm run start
```
